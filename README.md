# 🌈 MyMoji — 나를 나타내는 감정 이모티콘

> 하루의 감정과 성향을 기록하고, 나를 대표하는 이모티콘으로 표현하는 감정 기록 웹앱  
> **Frontend:** HTML · CSS · JavaScript · Firebase (Auth)
> **Backend:** Spring Boot · PostgreSQL

---

## 🧩 프로젝트 개요
신체건강을 넘어서 정신건강의 중요성이 커지고 있는 현대사회에서, 자기이해와 감정관리는 필수적입니다.<p> 
하지만, 자기자신을 판단하는 것은 쉽지 않고, 감정은 빠르게 휘발되기 때문에 ‘기록’이 필요한 지금인데요. 

**MyMoji**는 매일 간단한 기록을 하면서 사용자의 감정과 성격 데이터를 바탕으로  
사용자만의 이모티콘을 추천,자기이해와 감정관리 팁을 제공하는 프로젝트입니다.

사용자는 매일 간단한 6개의 질문에 답변하며 자신의 성향(정서안정성, 외향성, 친화성, 성실성, 개방성)을 업데이트할 수 있습니다.  
이후 서버에서 해당 데이터와 평균 점수를 기반으로 맞춤형 이모티콘과 이모티콘의 설명, 이모티콘 별 감정관리 피드백을 제공합니다.

---

## 🚀 주요 기능

| 구분                 | 설명                                                     |
| -------------------- | -------------------------------------------------------- |
| 🧑‍💻 회원가입 / 로그인 | Firebase Auth를 기반으로 구현                            |
| 🏠 홈 화면           | 상단 인삿말 + 오늘의 질문 6개 + 이모티콘 결과 표시       |
| 💬 질문 응답         | 말풍선 클릭 → 모달에서 1~5점 슬라이더 선택               |
| 💾 이모티콘 확인       | 설문 제출 후, 이모티콘과 이모티콘 설명, 감정관리 피드백 확인 |
| 📅 기록 탭           | 날짜/시간별로 생성된 이모티콘의 변화를 확인                |
| ⚙️ 설정 탭           | 닉네임 및 사용자 정보 관리 (추가 예정)                   |

## 🧭 동작 흐름

1️⃣ **로그인 / 회원가입**  
→ Firebase 인증 or API `/api/users/signup`

2️⃣ **질문 불러오기**  
→ `GET /api/questions/daily`  
→ 6개의 질문 랜덤 수신 → 말풍선 UI에 표시

3️⃣ **답변 입력 (1~5점)**  
→ 각 말풍선에 점수 저장 (`data-qid`, `data-score`)

4️⃣ **응답 완료 시**  
→ `{questionId, score}` 6개를 한꺼번에 백엔드 `/api/answers/daily`로 전송  
→ 백엔드에서 이모티콘 결정 후 반환

5️⃣ **기록 탭 반영**  
→ 데이터베이스 동기화 후 기록 탭에 반영
